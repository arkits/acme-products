## Acme Data Products (client-only)

React + Vite app for creating and sharing data products across Acme Corp. Client-side only with localStorage persistence.

Features:
- Create a data product (name, description, line of business)
- Link data sources (dataset with AVRO, API with OpenAPI/Swagger)
- Visualize aggregate objects from all data sources
- Show sample SQL and curl for consumers
- Dark UI with Tailwind CSS

Getting started:

```bash
npm install
npm run dev
```
